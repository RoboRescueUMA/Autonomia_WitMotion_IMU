## INSTALACIÓN

```bash
sudo apt-get install libqt5serialport5-dev

cd catkin_ws

git clone --recursive https://github.com/ElettraSciComp/witmotion_IMU_ros.git src/witmotion_ros

catkin_make
```


## USO DIARIO

`roslaunch witmotion_ros witmotion.launch`
